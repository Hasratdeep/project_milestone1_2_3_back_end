# Endpoints implemented:

- POST /tasks → create a task (createTask)
- GET /tasks → list tasks (getTasks)
- GET /tasks/:id → get a single task (getTask)
- PUT /tasks/:id → update a task (updateTask)
- DELETE /tasks/:id → delete a task (deleteTask)
